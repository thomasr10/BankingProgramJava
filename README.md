# 🏦 Mini Banking Program

**Mini projet pour continuer d'apprendre le Java.**  
Ce petit programme de console simule un système bancaire simple permettant de consulter le solde, déposer ou retirer de l’argent.

## 💡 Objectif pédagogique

Ce projet a pour but de pratiquer les concepts suivants en Java :

- Utilisation de la classe `Scanner` pour lire les entrées utilisateur  
- Boucle `while` pour maintenir le programme actif  
- Structure de contrôle `switch` pour gérer les choix de l’utilisateur  
- Méthodes `static` pour organiser le code et éviter les répétitions  

## ⚙️ Fonctionnalités

- 🧾 **Afficher le solde** actuel  
- 💰 **Déposer** de l’argent  
- 💸 **Retirer** de l’argent (avec vérification du solde)  
- 🚪 **Quitter** le programme proprement

## ⚙️ Installation & exécution

1. **Cloner le dépôt :**
   ```bash
   git clone https://github.com/thomasr10/BankingProgramJava.git
   cd BankingProgram
   ```
2. **Compiler le programme:**
   ```bash
   javac Main.java
   ```
3. **Lancer le programme**
   ```bash
   java Main
   ```
